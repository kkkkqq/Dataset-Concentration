# Dataset-Concentration
The official implementation of the dataset concentration method.
Please refer to the 'run_script' folder for the example shell scripts for synthesis and doping. The scripts should be executed in the order of their labels.
Please download the corresponding DiT model and VAE checkpoints from the following pages:
- DiT: https://github.com/facebookresearch/DiT
- VAE: https://huggingface.co/stabilityai/sd-vae-ft-mse
The VAE checkpoints shall be stored in the 'sd-vae-mse' folder.

Please consult the authors if you have further questions.
